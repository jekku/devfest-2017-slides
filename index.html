My 'Go To' interview question 

              
                'use strict'

                const numbers = [1, 2, 3, 4]
                [1, 3] // Even numbers filtered out
                [2, 4] // Odd numbers filtered out
                [2, 4, 6, 8] // List values are doubled
                2 // find the first even number you encounter
                1 // find the first odd number you encounter
                24 // Multiplied from left to right (Row multiplication)
